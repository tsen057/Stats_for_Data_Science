Task 1: Examine the data

Q Now open the data (say in Excel), examine it and answer the following questions: The data is obviously not pre-processed – can you spot the problems?

Following problems can be spotted in the data -

  1. The data is structured in such a way that it is readable for humans. The same data cannot be read by the computer easily.

  2. It consists of empty cells indicating null values. For example for Sensible heating in large offices is not present.

  3. It has duplicate data, having identical rows of data, such as at for Indoor Temperature at peak.

  4. We can see certain values where the data is significantly different from the rest of the data. It can be seen in case of heating energy in large offices of El Paso.

Q How would you address the challenges identified above? Briefly summarize your thoughts.

We can use following approaches for handling above problems -

  1. The structure of the data can we changed using some tidyr package functions like gather and mutate.These will help in creating the data which can be read by computers.

  2. Null or empty values can cause deviation in the result of data analysis. To avoid that we can fill the empty values with approximate values of mean or median, which is called as Imputation technique.

  3. For duplicate data, aggregation of data can be beneficial. It would create an aggregate temperature value that will reduce multiple entries.

  4. We can see some extreme values in data known as ‘data outliers’. For addressing them, we can either remove the values or transform the data with the help of mathematical and statistical techniques.

For small office with sensible cooling in place, how does the outdoor temperature at peak load differ by weather? Does the pattern vary by office type?

dataframe3 %>% filter(office_size == 'small') %>%
  filter(description == "Outdoor Temperature at Peak Load [C]" & group=='sensible_cooling') %>%
  ggplot(aes( x = weather,y = value,color=weather,shape = office_type)) + geom_point(size = 3) +
  theme(axis.text.x =element_text(angle = 40, hjust = 1),
        legend.position="right",
        legend.text = element_text(size=6),
